Tree Node Views

Each node in the results tree has a view associated with it. The typical views are:

  • Table
  • Graph
  • Dynamic Results
  • Template
  • None

Table View

Selecting a node which is associated with a table view will display a table on the right side. This table may contain multiple rows and columns. You can manipulate this view by resizing the columns and reordering them. Double-click the divider to the right of a column to automatically resize the column to fit the heading or contents. Click on a column header to sort the table by the contents of the column.

Use the Filter Toolbar or right-click anywhere on the table to get the Table shortcut menu to filter the table view or copy cells.

See also Applying Templates.

Graph View

Use the Filter Toolbar or right-click anywhere on the graph to get the Graph shortcut menu to filter the graph view.

View a graph created in Spirent TestCenter (including graphs with a logarithmic axis or multiple axes) by clicking on the graph node in the tree, or create a graph.

To create a graph:

Select a node in the results tree that displays a table view on the right side.

The Create Graph icon will become accessible.

The data for the chart comes from the selected table view. You can use all the data from the table or a subset. The entire table is used if no selection is made.

To select a subset of the data in a table:

Define the area of the table that contains the data you want to graph, by either one of these techniques:

  • Press and hold the left mouse button while dragging through an area of the table.
  • Select a cell in the table, then hold down the SHIFT key while selecting another cell.

After selecting the data to graph, click the Create Graph icon.

The Configure Graph window will open. Use this window to select:

  • The type of graph (bar, line, or pie)
  • The perspective (2D, 2.5D, 3D)
    • 2.5D uses three-dimensional elements. Values in different table rows are displayed in a single row on the graph.
    • 3D uses three-dimensional elements. Values in different table rows are displayed in different rows in the graph.
  • Whether to interpret the series from the table in rows or in columns

When you have finished configuring the graph, click OK. The graph will be added to the results tree with the name untitled, in a folder named Graphs. (If not already present, the Graphs folder will be added to the results tree under the current results database.)

To rename the graph, double-click on the name in the results tree, type a new name, and press ENTER.

A 3D graph can be manipulated by dragging the mouse while pressing a button:

Button Effect on the graph
Left Rotate
Right Move
Middle (or left+right) Zoom

To close a graph

Highlight the graph in the Results Reporter tree and press DELETE.

Dynamic Results View 

View a dynamic results view created in Spirent TestCenter or create a new dynamic results view with the Dynamic Results View wizard.

Customize a dynamic results view with the Dynamic Results View

Template View

A template node in the results tree can contain any of the other node types. Toolbar buttons and menu items pertaining to reporting are enabled when viewing a template view. Nodes subordinate to template view nodes use the default view of the node type, e.g., table nodes use table view and chart nodes use chart view.

Drag and drop a column in a rendered table to change the table layout.

See also Reporting.

None

Selecting a node which is not associated with a view will clear the display on the right side. Usually these nodes function to organize the results tree and are only for navigational purposes.
