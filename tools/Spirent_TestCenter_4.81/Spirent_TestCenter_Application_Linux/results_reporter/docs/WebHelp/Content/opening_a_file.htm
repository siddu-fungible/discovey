Opening Results Files

Results Reporter can open one or more SQLite databases in several ways.

To open a file:

  1. Select File > Open > Results Database.
  2. In the Open dialog box, select the file type you want to open, a Timed Result Log (.trl) or a Spirent TestCenter Results Database (.db).
  3. Navigate to any folder on the local hard disk or to folder within any mounted network drive, and select one or more files. Hold down the Shift key to select a range of files or hold down the CTRL key to select multiple files individually. The names of all selected files are displayed in the File Name area.
  4. Click Open to load the selected files into Results Reporter.

Results Tree

The tree display on the left side of the screen shows each of the currently open result databases. Opening additional result databases adds new nodes to this tree. The path to the result database is displayed as a label on the node.
