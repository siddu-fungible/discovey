Spirent TestCenter Release Notes and New Feature Information

Release Notes and New Features documents are available for download on the Spirent Customer Service Center (CSC) website (https://support.spirent.com/SpirentCSC/).

These documents are located in the Related Resources list on the CSC application Downloads page.

Select the Release Notes or New Features document whose version matches the Spirent TestCenter software you have downloaded, or are about to download.

To retrieve Release Notes and New Features documents:

  1. Navigate to https://support.spirent.com/SpirentCSC/.

  2. Log in to the Customer Service Center.

    NOTE: You must be logged in to access Release Notes and New Features documents.

  3. Click the Downloads icon on the left panel of the screen.

  4. Choose General Release in the Release Type box.
  5. Choose Spirent TestCenter in the Product line selection box.

  6. Choose Applications in the Categories list.

    You can also filter by release number. Select the release number from the option box (this figure shows 4.64 as an example).

  7. In the Applications list, scroll to the application for your operating system (this example shows the Windows application).

  8. Click Related Resources to open the list.

  9. Click the document title link.
