Using the Online Help

The online help includes buttons along the top that display left–hand panes, toolbars, and a search input field. Each topic also contains various methods for navigating. The online help system uses a multi–platform HTML–based output, which is designed to open in your default Web browser. This Help system resides on your computer and is installed during the TestCenter software setup process.

Navigation buttons

The online help includes the following left–hand panes. To access each pane, click the Contents, Index, Search, or Glossary button on the toolbar.

Pane Description
Contents The Contents pane displays the table of contents books and pages that represent the categories of information in the online Help system. Clicking a closed book opens the book to display its content (sub–books and pages). Clicking an open book closes it. Clicking pages selects topics to view in the right–hand pane.
Index The Index pane displays a multi–level list of keywords and keyword phrases. These terms are associated with topics in the Help system and are linked to specific topics. To open a topic in the right–hand pane associated with a keyword, click the keyword. If the keyword is used with more than one topic, a dialog opens, from which a specific topic can be selected to view.
Search When key words are typed into this Help system's Search pane, a list of related topics are displayed. Click a topic to display it in the right–hand pane.
Glossary The Glossary pane displays a glossary similar to those found in printed publications. It provides a list of words, short phrases, and their definitions related to TestCenter and/or the AX/4000. When a term is selected from the Term pane, its corresponding definition is displayed in the Definition pane.

Other buttons and toolbars

Feature Description
Search field To perform a search, type a word or phrase in the Find box in the toolbar. Then, click Go.
Hide the left pane To provide a larger viewing area of the open topic, hide (or close) the left pane. To hide the left pane, click the x button. To show the left pane again, click the Contents, Index, Search, or Glossary button.
Browser toolbar The toolbar at the top of the window is the standard toolbar for your default browser. Use it to go back and forward through the history of visited topics, print a topic, or perform other tasks.

Using context–sensitive Help

While working in TestCenter, display information about windows or dialogs by using the context–sensitive Help available in the application. Many dialogs include Help buttons. Click the Help button and a topic opens in a custom window that explains how to use the fields and controls at the dialog.
