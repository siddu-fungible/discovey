What Is Results Reporter?

Spirent TestCenter Results Reporter is a reporting tool that you use to analyze and display data generated by Spirent TestCenter. Results Reporter displays test results in tabular and graphic formats that you access through a simple user interface.

You see an empty screen when you start Results Reporter. Note the menu bar and toolbar at the top of the window, the Results tree on the left, and the main display area on the right.

Information Saved by Spirent TestCenter

Spirent TestCenter stores information generated both during and after a test. This information is composed of statistics, configuration, and views. All this information is saved in an SQLite database.

A test may contain single or multiple iterations. A single SQLite database contains all information across a test run, whether it be a single iteration or hundreds.

The Results Reporter can open, display, and analyze any number of SQLite databases. So you can open results from multiple test runs and view them simultaneously.
