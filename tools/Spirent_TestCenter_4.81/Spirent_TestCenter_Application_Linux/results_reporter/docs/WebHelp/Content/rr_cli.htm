Command Line Interface

Results Reporter provides a command-line interface to generate reports without running the GUI.

Command

Linux: ResultsReporterCLI.sh

Windows: ResultsReporterCLI.bat

Parameters

Parameters specify the source file, destination file and format of the report. There are terse and verbose versions of each parameter. Entering the command with no parameters displays the command-line help.

Option Description

-d <file>

--dest <file>

The target path and file name for the report

-e <subject>

--email <subject>

 Send the report as an email with the indicated subject

-f <pdf|pdf-tree|html|csv|csv-tree|jpeg>

--file <pdf|pdf-tree|html|csv|csv-tree|jpeg>

The file format for the report

To export iterations, specify the iteration database file name with -d or --dest option. Specifying the summary database file name will not cause iteration databases to be exported.

Tree-based formats (pdf-tree, csv-tree) cannot use templates (the -t or --template options).

-h

--help

 Command-line help

-i <index>

--StartIndex <index>

 Starting value of the index value for the table export

-m <database-list>

--merge ,database list>

 Combine several databases into one

-n <count>

--ExportRowCount <count>

 Total row count of the exported table

-o <file>

--open <file>

The source path and file name for the results file (.db)

-p <smtp-server>

--mail-server <smtp-server>

 Hostname of the mail Server

-r <recipient-addresses>

--recipients <recipient-addresses>

 Recipients of the email report

-s <port>

--server <port>

 Start a web server on the indicated port

-t <file>

--template <file>

The source path and file name for a template used to interpret and format the report

Not used for tree based formats (pdf-tree, csv-tree)

-u <sender-address>

--from <sender-address>

 Sender for the email report

-x

--GetTableRowCount

 Returns the total row count for all template tables

Example

In Linux, to generate a PDF report from MyResults.db, using the template CustomStats.rtp, and write the report to MyResults.pdf:

./ResultsReporterCLI.sh -o MyResults.db -f pdf -d MyResults.pdf -t templates/CustomStats.rtp

In Windows, to generate a PDF report from MyResults.db, using the template CustomStats.rtp, and write the report to MyResults.pdf:

ResultsReporterCLI.bat -o MyResults.db -f pdf -d MyResults.pdf -t templates/CustomStats.rtp
