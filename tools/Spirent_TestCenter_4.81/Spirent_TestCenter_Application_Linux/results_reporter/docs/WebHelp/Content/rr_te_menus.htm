Results Reporter Template Editor Menus

The Results Reporter Template editor provides the following commands accessible via the Template menu or the toolbar. Position the cursor over an icon for a few seconds and the tooltip identifier for that icon will display.

Icon Keyboard Shortcut Meaning
CTRL+D

Open Database
CTRL+E Open Template
CTRL+S

Save Template

CTRL+P Generate PDF
CTRL+H

Generate HTML

CTRL+R Requery All
    Use the dropdown box on the right side of the toolbar to select between iterations of a multiple-iteration database
