Template Editor Reported Stats Tab

Use the Reported Stats tab to extract statistics from one or more database tables to use in a computed statistic or a template table or graph. Double-click a field to edit the value.

Field Description
Query SQL query to generate the statistic
Name

Unique user-defined name for the statistic

This name is used on the Tables and Graphs tabs

Value

Read-only display of the results of the query

Multivalued

Clear to indicate the statistic contains a single value

Select to indicate the statistic contains multiple values

Cell Value

Displays the selected cell value from the reported stats table

Click on To display
Query Editable query
Value Display-only value of the statistic

 

Button Description
New

Add a statistic to the bottom of the list

Double-click the Query field to enter the SELECT statement for this statistic

Double-click the Name field to enter a name for this statistic to be used in the Tables and Graphs tabs

Add A Reported Statistic

Wizard to add a statistic

Option Description
Choose a table Select a table from the database
Choose a column

Select one or more columns from the table

Use SHIFT click and CTRL click to select multiple columns

Remove Remove the highlighted statistic from the list
Update Update the Value column based on the query
