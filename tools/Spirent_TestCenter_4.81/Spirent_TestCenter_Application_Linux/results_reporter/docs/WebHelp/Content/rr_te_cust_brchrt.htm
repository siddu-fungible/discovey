Template Toolbar: Valid Bar Chart Control Values

Background: Reporter Menus and Toolbars (see Bar chart controls); Bar chart controls (see Show Preferences - Charts)

Validation rules apply to the From / To or Sampling Rate values that you enter to control a bar chart display.

  • Values must be whole numbers.

    Do not enter a decimal, fraction, negative number. Do not enter non-numeric characters.

  • The From value must be greater than or equal to 0, and less than or equal to the To value.
  • The To value must be greater than or equal to the From value, but no greater than Bar Chart Total Data - 1.
  • If the From or the To value is invalid or blank when you click Refresh, Spirent TestCenter populates the field with the default value.
  • The Sampling Rate value must be equal to or greater than 1, and must not exceed the total number of chart columns.
  • If you enter a decimal Sampling Rate value and click Refresh, Spirent TestCenter discards the numbers following the decimal point.
  • If the Sampling Rate value is invalid or blank when you click Refresh, Spirent TestCenter populates the field with the last validated value entered.

TIP: Every time you change a range or sampling rate value, you must click Refresh to apply the change to the bar chart display.
