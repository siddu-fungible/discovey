Bar chart controls

 

Use the bar chart controls in the toolbar above a Test Summary page to customize the data display. You can control column density, or drill down to a specific range of columns.

TIPS:

  • To add bar chart controls to the Template toolbar, select an option on the Charts tab in the Results Reporter Preferences window.

    The option you select determines which bar chart control option appears in the toolbar. See Tools > Show Preferences > Charts in the Menus table in this topic.

  • To enable the bar chart controls you must select a Test Summary page that contains a bar chart.
  • Enter range values or a sampling value, and then click the bar chart Refresh button to apply the value(s).
  • Settings apply to the current bar chart, only. When you select a different Test Summary page with a bar chart, the display control values you entered revert to default. You must re-enter values for the new view.
Selection in Preferences Displayed in the Template toolbar
Using display range (from - to)

Bar Chart Total Data: read-only

Display from... to: To drill down to a more detailed view of the data, enter a starting column number and ending column number. For example, enter (From) 7 and (To) 20 to display columns 7 through 20 in the bar chart.

Default:

To: Bar Chart Total value - 1

From: Bar Chart Total value - 1

See Template Toolbar: Valid Bar Chart Control Values for important information about valid values.
Using sampling rate

Bar Chart Data Sampling Rate (1/x): Enter an index number to set sampling for the display. For example, if you enter 7 the bar chart displays every seventh column (the index is 0, 7, 14, 21, etc.)

Default: 1

Bar chart refresh.

Click the Refresh button in the bar chart section of the toolbar to apply your range values or sampling rate values to the bar chart display. See Bar chart controls for additional information.

TIP: Every time you change a range or sampling value, you must click Refresh to apply the change to the bar chart display.

Table Shortcut Menu

Right-click on a grid in the right pane to bring up the shortcut menu.

Option Description
Filter

Open the Configure Filter window to set parameters for filtering rows displayed in the grid.

To set a filter:

  1. Select a field in the left pane.
  2. Select the Selected field to include this field in the filter.
  3. Configure the condition by selecting a logical operator (GREATER THAN, GREATER THAN OR EQUAL TO, LESS THAN, LESS THAN OR EQUAL TO, EQUAL TO, NOT EQUAL TO, or BETWEEN) and the value or values required to complete the condition. The available operators will vary based on the data type of the field.
  4. Repeat steps 1-3 for each field to be used in the filter.
  5. If two or more fields are selected, click the Filter Options button to connect the conditions with AND or OR.

For tables based on the results database (but not for template tables), you can create a query based on a SQL command. Select From Query to indicate the filter is based on a SQL query. Click Update SQL to generate the SQL statement that reflects the current settings, then edit that statement for the desired query.
