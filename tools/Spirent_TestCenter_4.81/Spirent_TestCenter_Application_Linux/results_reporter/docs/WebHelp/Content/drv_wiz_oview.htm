Dynamic Results View Wizard

Use the Dynamic Results View wizard to create a dynamic results view node.

  1. Enter a name for the view.
  2. Select Port, StreamBlock or External Device from the dropdown list, then select objects for the view.
  3. Select result properties and add to result view display.
  4. Create filter condition and add into the summary.
  5. Setup how you want to group and sort the results.
  6. Limit the number of records returned by the query.
