Spirent TestCenter Results Reporter

What Is Results Reporter?

Spirent TestCenter Results Reporter is a reporting tool that you use to analyze and display data generated by Spirent TestCenter. Results Reporter displays test results in tabular and graphic formats that you access through a simple user interface.

You see an empty screen when you start Results Reporter. Note the menu bar and toolbar at the top of the window, the Results tree on the left, and the main display area on the right.

Information Saved by Spirent TestCenter

Spirent TestCenter stores information generated both during and after a test. This information is composed of statistics, configuration, and views. All this information is saved in a set of SQLite databases. A summary database contains summarized results for the entire test. An detail database contains test result details. In a multi-iteration test, each iteration is stored in a separate database.

Results Reporter provides the tools to read the data and format it into useful reports. (Results are generated by Spirent TestCenter, but cannot be loaded into Spirent TestCenter.)

You can use Results Reporter to view results as the test is running. Results Reporter will open the database in read-only mode while Spirent TestCenter is writing data to the database. Once the test is completed, Results Reporter will open the database in read-write mode. In this mode, graphs and queries can be created and saved with the results.

Results Reporter can open, display, and analyze any number of results databases. So you can open results from multiple test runs and view them simultaneously.
