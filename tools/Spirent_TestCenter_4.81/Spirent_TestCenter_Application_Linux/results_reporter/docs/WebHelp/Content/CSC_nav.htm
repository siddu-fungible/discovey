Spirent Knowledge Base: Documentation

Additional Spirent TestCenter documentation is available on the Spirent Knowledge Base website. You can retrieve specific documents, or search the Knowledge Base to retrieve relevant information from a variety of sources. The following titles are among the many Spirent TestCenter information sources available on the Knowledge Base:

  • Spirent Hardware Reference: detailed information about Spirent equipment.
  • Spirent TestCenter Virtual quick start guides: explain how to set up the virtual controller, and install virtual licenses.

  • SPT-N11U chassis accessory instructions: explain how to use the card carriers and how to replace the fan tray.

Search by document title or keyword, or browse user documentation titles.

To search for a document:

  1. Navigate to https://support.spirent.com/SpirentCSC and log in to the CSC.

  2. Click the Knowledge Base tab in the navigation pane on the left.

  3. Type keywords or a full document title into the Search box, and click Search.

    The Results pane displays links to relevant titles.

    TIP: Select one or more filters in the left pane to narrow your search results.
  4. In the Results pane, scroll to the name of the document you want to view and click the link.

To browse user documentation:

  1. Navigate to https://support.spirent.com/SpirentCSC and log in to the CSC.
  2. Click the Knowledge Base tab in the navigation pane on the left.

  3. In the Browse by Article Type navigation pane, click Documentation.

    A list of documents appears in the pane on the right. The Categories navigation pane appears below the Browse by Article Type pane.

    TIP: In the Categories pane, select items from the Document Type and Product lists to narrow your results.
  4. In the Results pane, scroll to the name of the document you want to view and click the link.
