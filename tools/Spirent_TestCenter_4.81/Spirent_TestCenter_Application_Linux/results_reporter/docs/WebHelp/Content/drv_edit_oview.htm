Dynamic Results View Editor

Click on the Customize toolbar button in a dynamic result view node to open the Dynamic Result View editor to customize a dynamic results view node.

Tab Description
Objects

Select the checkbox for the objects to use in the view.
Columns

Use the Node Filter to filter out nodes in the selection tree.

Highlight a field and click Insert or Remove to maintain the list of columns for the view.

Arrange the order of the columns with the Move Up and Move Down buttons.

Option Description
Counter Counter results
Config Configuration parameters
Conditions

Click the Add Filter or Delete Filter button to add or delete a row to the top window.

Option Description
Property Name List of properties selected in the previous step or tab.
Condition A logical condition for comparison of the property to the value.
Value A property name or a static value to which the property is compared.
Relation A boolean operator to join filters.

Click the Add Filters to Condition button to move the filter from the top window to the bottom window.

To edit a condition, highlight it in the bottom window. It shows in the top window, where it can be edited. Click the Update Condition button to update the condition in the bottom window.

Highlight a condition in the bottom window and click the Delete Condition button to delete a condition.
Grouping/Sorting

Group and sort options

Option Description
Property List of properties selected in an earlier step or tab.
Caption A user-defined value to identify the property.
Group Select the checkbox to group by a property. If the checkbox is disabled, that property cannot be used for grouping results.
Sort Select a sort order to sort the results on a property.
Limit

Restricts the number of records returned by the query.

Setting a value larger than recommended can affect application performance.
