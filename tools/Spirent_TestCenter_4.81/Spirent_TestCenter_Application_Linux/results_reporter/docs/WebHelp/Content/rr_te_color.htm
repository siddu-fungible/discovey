Template Editor Color Codes

The template editor uses standard HTML color codes to specify the color of an element. A color code is a three-byte value, expressed in hexadecimal, that specifies the amount of red, green and blue in the color, from #000000 (black) to #FFFFFF (white). For example, #FF0000 is the brightest red possible, #00FF00 the brightest green, and #0000FF the brightest blue. Other colors are indicated through different combinations of red, green and blue.

Code Color
#FF0000 Red
#00FF00 Green
#0000FF Blue
