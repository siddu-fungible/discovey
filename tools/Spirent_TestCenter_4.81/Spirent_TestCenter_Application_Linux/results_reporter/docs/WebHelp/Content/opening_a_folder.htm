Opening a Folder of Files

Results from multiple test runs are often located in a single folder. You can open all the results files in a folder using the Folder of Files option on the Open menu.

To open a Folder of Files:

  1. Select File > Open > Folder of Files.
  2. In the Open dialog box, select a folder on the local hard disk or mounted network drive.
  3. Click Open. Results Reporter attempts to load every SQLite database that is located in the selected folder and its subfolders. If the folder contains files that are not valid SQLite database files, they are not loaded and an error is displayed.
