Template Editor Graphs Tab

Use the Graphs tab to create 2D and 3D line, bar and pie graphs. Use the Render button at the bottom of the window to view the graph after you configure the values. Right-click the X or Y axis on the graph to see the Graph Shortcut menu.

Graphs

The top section of the window shows the currently defined graphs. Use the Add and Remove buttons to add a graph to the bottom of the list or remove the highlighted graph from the list. Double-click a field in the list to change the value.

Field Description

Background Color Color of the area outside the graphed values
Plot Color

Set the color of the plotted values in the Graph Properties window.
