Template Editor Layout Tab

Use the Layout tab to arrange elements on a page. The window is divided into three sections, the Pages tree, the Properties window, and the page view, where the page is rendered.

Pages Tree

Button Description
Insert Front Page

Create a standardized, customizable, front page for the report

By default, the front page contains a MultiText field that displays the contents of the computed stat RRFrontPageText. This variable combines information that may be saved in the results database, such as test duration, test state and user notes.

To include information in the results database to be displayed in this field, use the Spirent TestCenter File Properties window (File menu, Properties item) to create name-value pairs.
Add Page Add a blank page to the end of the template. Right-click a page to move it up or down in the list.
Add Content

Add a text box, a MultiText field, an image, a graphic or a table to the highlighted page

Enter a name to identify the element under the Contents node in the Pages tree

A table can also be added to a page by dragging it from the Results Reporter tree onto the page

Note: Only a table grid with a single query can be added.
Remove

Remove the highlighted page

Properties Window

Highlight a page element to view the properties. Highlight a page node to see the page properties.

Position

All page elements have position properties expressed in pixels. (Pages do not have position properties.) When an item is selected, rectangles appear on the corners. The element can be positioned by dragging it, which changes the X and Y values. However, the rectangles are not handles and cannot be used to resize the element. Resize an element by setting the values in the Width and Height fields.

Field Description
X Vertical distance from the top of the page
Y Horizontal distance from the left of the page
Width Width of the page
Height Height of the page

Link

A link connects a page element to another template page. Links can be created from a Text, MultiText, Image or Graph element. Template page names are case-sensitive, as is the content of the Link field. A link can also be a computed statistic. To link to a page name that will be computed reference the computed statistics in the Link field in the form =$NameOfStatistic$. The value of NameOfStatistic is evaluated at runtime and rendered in the path in the Location field.
