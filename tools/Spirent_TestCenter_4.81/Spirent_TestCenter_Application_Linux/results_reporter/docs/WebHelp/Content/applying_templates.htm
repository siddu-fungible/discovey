Applying Templates

Templates can be applied to any set of test results. To do so, click on a results database node (a child of the top level Results node) to get a drop-down list in the right window.

Choose from a list of available templates. The choice of template will dictate the pages displayed in the Pages dropdown list.

Use the template navigation toolbar to move between pages in the template.

Right-click a rendered table in the right window to open the template table shortcut menu.

NOTE: Templates are located in a folder named templates in the same directory where Results Reporter is installed. Please contact Spirent Global Services to learn about adding templates that are customized to your needs.

After adding a template, you must close and restart the Results Reporter for the new template to be available.
