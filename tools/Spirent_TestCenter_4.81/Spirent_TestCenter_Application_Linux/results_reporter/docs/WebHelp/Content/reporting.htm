Reporting

Reporting is available when viewing a results tree node with a template associated to it, or when applying a template. In such cases the reporting menu items and reporting toolbar icon are enabled. Reports can be created in either PDF or HTML format.

PDF Reports

To create a PDF report, without customization:

  1. Click the Generate PDF Report icon in the toolbar or choose Generate PDF from the Report menu.
  2. If the selected node is not a template node, use the checkboxes to select the items to be included in the report. The export is restricted to the selected iterations. Selecting the root node (Everything) will not export the entire tree. Each node is displayed on a separate page.
  3. Click OK.
  4. Specify a folder and file name for saving the PDF report file.
  5. Click Save.
  6. If a page footer is desired, select the Enable Footer checkbox and then specify the content for each section of the footer. Click Continue.

Custom PDF Reports

Customized reports can be created for supported protocols. Supported protocols are listed in the Spirent TestCenter New Features document for the current release. See Spirent TestCenter Release Notes and New Feature Information for navigation instructions.

You define the basic contents of a custom report by selecting a result database. After you select your database, you choose the chapters and data items to include in the report.

To create a custom PDF report:

  1. To populate the Results Reporter navigation tree, and enable the Generate Custom Report option, open at least one result database for a supported protocol.

    1. Click the Open Result Database icon in the toolbar.

    2. Select a database file for a supported protocol.

    3. Click Open.

      The results database file navigation tree appears in the pane on the left side of the screen. The default Results Reporter report template appears on the right side of the screen. (This is not the template used in the custom report.)

  2. In the Results Reporter menu bar, click Report > Generate Custom Report.
  3. Follow the wizard prompts to set up your custom report.
    1. Select one or more supported results database files, and then click OK.

    2. Click to select chapters and data items for your report, and then click OK.

      If you want to include all database content in your report, click the checkbox next to Everything.

    3. Select the folder where you want to save your PDF report, and then click OK.

      Results Reporter generates your report and saves it to the specified folder.

  4. Navigate to the folder where you stored your new report, and open the report.

HTML Reports

To create an HTML report:

  1. From the Report menu, choose Generate HTML.

  2. Navigate to the folder where you want to save the HTML report file, and then click Save.
